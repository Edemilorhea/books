# 為什麼男人愛說謊 女人愛哭 - 艾倫·皮斯（Allan Pease）、芭芭拉·皮斯（Barbara Pease）

**分類標籤**：#兩性溝通 #性別差異科普 #親密關係 #溝通技巧 #演化心理學 #通俗心理學

**一句話核心主張**：男女大腦運作方式與溝通習慣有其演化與生理根源（女性多軌／情感導向，男性單軌／解決問題導向），唯有學會辨識並使用對方的「語言」，才能化解嘮叨、情緒勒索、評分落差等關係中常見的破壞性互動模式。

## 核心框架/方法論

- **大腦結構差異**：作者引用（有爭議的）腦部掃描研究，主張女性大腦語言區「多軌式」（可同時處理多話題、多工），男性大腦「單軌式」（一次只能專注一件事）。此為全書解釋大多數兩性摩擦的核心前提。
- **嘮叨的本質**：嘮叨並非女性天生的性格缺陷，而是「直接表達需求失敗後」演變出的**迂迴、重複、暗示性**溝通模式；其結果是雙方互相「訓練」——受罪者的敷衍/逃避訓練了嘮叨者不斷重複，嘮叨者的重複也訓練了受罪者築起更高的心牆，形成惡性迴圈。
- **情感勒索六要素模型**：受害人（有罪惡感的一方）→ 勒索人（知道對方弱點）→ 提出要求 → 威脅（揭密／懲罰／製造愧疚）→ 對方反抗 → 對方妥協，迴圈持續強化。哭泣在此框架下被定義為人類獨有的「情緒操控工具」之一。
- **秘密評分法（隱形記分卡）**：女性傾向為伴侶的**每一個微小舉動**（無論大小）打分，重影片率與心意；男性傾向依**事情的客觀大小／金錢價值**打高分。雙方評分邏輯的系統性落差，是許多伴侶「明明都有付出卻互相不滿」的根本原因。
- **「男人抓狂七件事」與「男人七大謎」**：以演化心理學（獵人 vs 採集者、單一配偶制的生物稀少性等）逐一解釋看似惱人的男性行為背後的生物/歷史成因，再提供對應的溝通技巧。

## 精華技巧/可操作重點

### 化解嘮叨
- 用「我」陳述句取代「你」陳述句：先講對方的**行為** → 你的**想法** → 你的**感覺** → 對你的**影響**，避免對方進入自我辯護模式。
  - 範例：「這禮拜你都很晚回來，都沒打電話告訴我（行為）。我會胡思亂想，是不是你不在乎我了（想法），這讓我很難過（感覺），也讓我很擔心你（影響）。」
- 給彼此一段「緩衝時間」：下班後給男人一段安靜發呆的時間，再進行需要深度溝通的對話。
- 只講一次＋清楚說明後果，然後言出必行（不追著收拾、讓對方自己承擔沒做事的自然後果，例如把亂丟的濕毛巾直接放進他自己的枕頭下，讓他親身感受不便）。
- 感謝與肯定伴侶為家庭付出的瑣事，能大幅降低對方的嘮叨頻率——嘮叨往往是「未被看見」的求認同呼喊。

### 破解情感勒索
- 識別特徵：對方用威脅（揭密、自殘、懲罰）搭配「你欠我」的愧疚感，逼你妥協。
- 可直接引用的回應話術：
  - 「哦，那是你自己的選擇啊。」
  - 「你正在氣頭上，等你氣消了，我們再談吧。」
  - 「我知道你不高興，但事情就是這樣。」
  - 「等你準備好了，我很願意和你好好談談。」
- 核心原則：不辯解、不用同樣的情緒勒索反擊、不因對方沉默或憤怒而讓步；把勒索者的無理取鬧當成小孩的哭鬧來對待——溫和但堅定地不妥協。

### 縮小評分落差
- 女性：接受男人重視「大事」的評分邏輯，主動鼓勵、獎勵他做的瑣事。
- 男性：牢記——一句讚美、一次主動洗碗、一朵不定期送的花，長期累積的加分效果，往往超過加班賺的錢或大禮物；重點是**頻率與心意**而非金額大小（連續6週各送一朵花＝6分，遠勝過一次送一大束只算1分）。
- 兩人評分落差 <15% 屬健康關係；15%-30% 代表關係緊張、有誤會；>30% 明顯有一方長期不快樂，需要正視。

### 面對男人的「自尊防衛」
- 避免任何聽起來像「質疑他能力」的建議，改用「我覺得⋯」句型而非「你錯在⋯」。
- 範例：迷路時說「這些街道好亂，我覺得問一下人比較好」而非「你根本不知道往哪走」。
- 找對路、做對事後記得明確稱讚，強化正向行為。
