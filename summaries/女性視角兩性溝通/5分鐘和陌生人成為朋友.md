# 5分鐘和陌生人成為朋友 - 唐·加博爾（Don Gabor）

**分類標籤**：#社交技巧 #破冰技巧 #肢體語言 #溝通框架 #人際關係
**一句話核心主張**：透過主動、真誠好奇與可複製的破冰話術（SOFTEN肢體法則＋禮貌性提問＋積極傾聽），任何人都能在幾分鐘內把陌生人變成朋友。

## 核心框架/方法論
- **SOFTEN 肢體語言法則**：Smile微笑、Open arms張開雙臂、Forward lean身體前傾、Touch握手、Eye contact眼神交流、Nod點頭——開口說話之前，肢體語言就已經決定對方是否願意跟你談下去。
- **自我表露四層次**：①禮貌問候 → ②背景資訊（工作／興趣） → ③觀點與喜好 → ④深層感受與重大人生經歷。關係越信任才往下一層推進，不可躐等，也不要一開口就掏心掏肺。
- **持續交談四關鍵**：①聚焦當下環境找話題 ②找出對方的「熱點」（他真正在乎、投入時間金錢的事） ③保持雙向資訊交換平衡（作者以「談話球」比喻一來一往） ④用閒聊試探、掌控話題方向。
- **TACT法則**（處理尷尬對話）：Think三思而後言、Actively listen積極傾聽、Consider考慮說出口後的結果、Tact注意表達方式。

## 精華技巧/可操作重點
- **破冰四招**：①禮節性封閉式／開放式提問（「你住哪裡？」「為什麼決定搬去那？」）②輕鬆評論眼前情境（聊天氣、聊場合）③真誠且簡短的讚美＋追問一個好答的問題 ④主動自我介紹，尤其利用談話出現「停頓」的瞬間開口。
- **記名字5秒訣**：①專注聽對方介紹，別分心 ②別想著自己接下來要說什麼，只管聽名字 ③立刻大聲複誦一次確認 ④聯想一個同名的熟人或名人 ⑤交談中與道別時再喊一次名字加深記憶；多人同時介紹時可用「字母鏈」（例如喬治George＋瑪麗亞Maria→聯想成GM）幫助記憶。
- **傾聽5技巧**：問說明性問題釐清細節、留意並追問對方主動透露的「無償資訊」、請對方舉例說明、聽出話裡沒說出口的「冰山資訊」（暗示但沒直說的話題）、適時總結重點確認理解無誤。
- **降低被拒絕的焦慮**：主動出擊的次數越多，累積的正面回應也越多；被拒絕通常不是針對你個人（對方可能只是正忙、心情不好），把注意力放在成功的回應次數上，而非拒絕次數。
- **自然結束對話**：在雙方都表達完想法、氣氛仍融洽時就主動總結收尾，避免尷尬冷場後才倉促逃離；具體約定下次見面時間（例如「下週一起看場電影，我打給你」）比空泛的「有空再約」更有誠意也更有效。
- **脫身糾纏／抱怨型物件**：先簡短回應表達同理（「聽起來你現在處境艱難」），再用「是／不是」或封閉式問題打斷對方的喋喋不休，取回話語權，最後禮貌握手、喊對方名字道別。
- **化解15種常見尷尬情境的共同原則**（如同事求代班、親戚逼問終身大事、被拉入辦公室政治口水戰、朋友只在訴苦時才聯絡你等）：先簡短表達理解或同理，再溫和但堅定地說出自己的界線，不必過度解釋或找藉口自責。書中特別提醒「找人幫忙時要直說，不要暗示、企圖操縱對方」。
